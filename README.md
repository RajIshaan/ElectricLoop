# ElectricLoop - E-Waste Marketplace

A minimal eBay-like marketplace for electronic waste trading with hyper tech animations.

## Features

- Frontend-only authentication (mock state)
- E-waste marketplace/shopping page
- Minimal, clean design inspired by eBay
- **Hyper tech animations** powered by Framer Motion
- Glass morphism effects
- Animated backgrounds
- Smooth transitions and hover effects

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Frontend-only** - No backend required

## Pages

- `/` - Home page with animated welcome screen
- `/login` - Sign in page with smooth animations
- `/signup` - Registration page
- `/shop` - E-waste marketplace with animated product cards

## Animation Features

- Floating gradient text
- Animated background blobs
- Staggered card animations
- Smooth page transitions
- Hover effects on all interactive elements
- Tech grid background pattern
- Glass morphism effects
